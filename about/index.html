About Pixel Icon Library

Introducing HackerNoon's Pixel Icon Library, an open-source collection of Pixelated Icons. Meticulously designed on a 24px grid, ensuring perfect alignment and consistency to enrich your web/app/product/page/life experience. Inspired by HackerNoon's retro design vibe, these icons capture the essence of the internet's good old days.

What's in the Pixel Icon Library?

  • 400+ Unique Pixelated Vector Icons.
  • Light SVG Files - 176 KB for the entire collection.
  • PNG files in 12px, 16 px, 24px, and 48px for both Light/Dark Mode
  • Multiple Ways to Use - Install via NPM Package, Directly via HTML & CSS, or via Figma Component Library.
